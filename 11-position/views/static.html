{ position: static; }

Les éléments HTML sont positionnés en static par défaut.
Ils se mettent les uns au dessus des autres ou côte à côte
Les éléments positionnés statiquement ne possèdent pas les propriétés top, bottom, left right et z-index.
Un élément avec { position : static; } n'est pas positionné de manière particulière ; il est toujours positionné selon le flux normal de la page :
